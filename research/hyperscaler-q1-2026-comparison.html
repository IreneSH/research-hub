Cloud 成長率分析

Google Cloud +63% 一枝獨秀，成長率為三家之冠，且連續加速（30%→63%），但須注意基數效應——GCP 營收僅為 AWS 的 53%。三家均表示需求持續超過供給，且 AI 帶動核心雲服務交叉銷售。

Azure +39% CC 在高基期下仍維持加速，且 Q4 FY26 指引 39-40% CC 意味著加速趨勢不減。MSFT 獨有的 Copilot 席位模型（20M+）使其 AI 變現路徑與純雲端不同。

AWS +28% 在 $150B ARR 基礎上令人印象深刻——這是 15 季最快增速，且自研晶片（Tranium + Graviton）提供了獨特的成本優勢。AI 帶動 core 增長的正相關性被首次明確量化。

CapEx 分析

三家 CY2026 CapEx 合計預計約 $560-580B——歷史性的投資規模。Amazon 領先（~$200B），MSFT 與 GOOGL 接近（各 ~$190B）。三家均強調投資由需求驅動，有 ROIC 框架支撐。

MSFT Q3 CapEx $31.9B 低於預期是本季最大驚喜之一，但 Q4 >$40B 指引顯示並非需求放緩，而是交付節奏波動。CY2026 ~$190B 含 ~$25B 元件漲價影響（主要記憶體）。

AMZN FCF TTM 僅 $1.2B 是三家中最受壓縮的，反映其最激進的投資週期。但 Jassy 以 AWS 第一波成長週期的經驗類比，認為收入成長終將超越 CapEx 成長。

GOOGL FCF 相對最健康（TTM $64.4B），但 2027 CapEx 「顯著增加」的模糊措辭仍為未知數。

Section 06

關鍵共同主題 & 差異化定位

Alphabet · 核心優勢

  • Cloud 成長率最快（+63%），AI Solution 收入 +800%
  • Cloud 利潤率改善最大（+15.1pp to 32.9%）
  • Search 查詢量創新高，AI 擴大廣告覆蓋率
  • Backlog $462B QoQ 翻倍，能見度最佳
  • 垂直整合全棧（晶片→模型→平台→應用）
  • UCP 商業協議建立 Agentic Commerce 生態

Microsoft · 核心優勢

  • AI ARR $37B 最大且增速 +123%
  • Copilot 20M+ 席位，使用強度 = Outlook
  • 營業利潤率 46% 三家最高
  • CapEx 低於預期 $3.4B，效率敘事最強
  • Seat + Consumption 模型最成熟
  • OpenAI IP 免版稅至 v32，結構性毛利優勢

Amazon · 核心優勢

  • AWS $150B ARR 基礎上仍加速至 +28%
  • 自研晶片業務全球前三（ARR $20B+）
  • Tranium 提供數百 bps OI margin 優勢
  • 廣告 TTM $70B+ 成為第三利潤引擎
  • OI margin 13.1% 史上新高（全公司層級）
  • OpenAI + Anthropic 雙模型平台化策略

三大共同主題

1. 需求 > 供給仍是核心敘事：三家均明確表示 AI 計算需求遠超可用容量，Cloud 收入「若無約束將更高」。這為 CY2026 合計 $560-580B 的 CapEx 提供了需求端支撐。

2. AI 利潤率反駁「燒錢無底洞」：GOOGL Cloud margin +15pp，MSFT 全年 OI margin +1pp，AMZN OI margin 創歷史新高——三家同時展現 AI 投資與利潤擴張並行的能力，正面反駁市場對 AI 投資侵蝕獲利的擔憂。

3. Agentic AI 成為下一個平台戰場：GOOGL 推 UCP 開放商業標準、MSFT 推 Copilot Cowork / Agent 365、AMZN 推 Bedrock Managed Agents + Rufus。商業模式從 per-seat 向 seat + consumption 演進。三家均視 agentic workflow 為未來幾年最大的成長向量。

半導體供應鏈總結觀點

CY2026 三家 CapEx 合計 ~$570B（GOOGL $185B mid + MSFT $190B + AMZN $200B），其中伺服器/短期資產佔比 60-67%，意味著 GPU、CPU、HBM、先進封裝、網路設備等的需求規模空前。

記憶體/HBM 為最大瓶頸：MSFT Q4 CapEx >$40B 中含 ~$5B 元件漲價，CY2026 含 ~$25B；AMZN 亦提及記憶體飆漲推動企業上雲。對 SK Hynix、Samsung、Micron 及 CoWoS 封裝供應鏈均屬正面催化。

自研晶片加速，但 NVIDIA 地位穩固：三家均維持 NVIDIA 為核心合作夥伴（GOOGL 首批 Vera Rubin、MSFT 持續大量、AMZN 持續大量），自研晶片主要用於推理降本而非完全替代。AMZN 自研晶片規模最大（$20B ARR），但強調「客戶要選擇權」。
