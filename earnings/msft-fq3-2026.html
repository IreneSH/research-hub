Section 06

Earnings Call Q&A 重點

Keith Weiss
Morgan Stanley
AI需求持久性？IT 預算從何而來？Seat + Consumption 轉型？
Amy：M365 正從純 per-seat 轉為「seat + consumption（meter）」模式，類似 Azure；bookings 形態將變化，但價值更大。Satya：所有 per-user 業務（生產力、編碼、安全）都將走向「user + usage」，由 agent 帶動的 workflow 壓縮創造客戶價值，驅動 IT 預算重新分配。
Karl Keirstead
UBS
CapExH2 CY26 需 ~$120B CapEx，物理供應鏈信心？
Amy：對供應鏈執行力有信心；H2 加速主要來自短期資產（GPU/CPU/Storage），元件漲價佔部分增量。1P usage（Copilot）在 Q3 進入不同軌跡；Azure Q4 指引 39-40% 已反映效率提升與需求兩面。
Brent Thill
Jefferies
利潤率AI 為何能提升利潤而非侵蝕？
Amy：AI 業務利潤率從一開始就優於當年 Cloud 轉型初期——① usage-based pricing 捕捉更多價值；② 合作夥伴 IP 免版稅使用；③ 自研硬體（Maia/Cobalt）持續降低 Infra 成本；④ SW/HW 雙重效率提升。
Gabriela Borges
Goldman Sachs
AICopilot 產品演進與 E7 策略？
Satya：Copilot 已從 chat → agent（researcher/analyst）→ edit mode → Cowork（非同步委派）全面演進；核心護城河是 Work IQ 17 EB 的組織脈絡資料，持續即時更新。多模型自動路由（含 Opus、Codex 等第三方模型）+ 情境豐富度 = 競爭者難以複製。
Kirk Materne
Evercore ISI
AIOpenAI 協議變更對財務影響？
Satya：整體滿意——① IP 免版稅存取至 v32；② OpenAI 仍為大客戶（AI + 其他計算）；③ 保有股權權益。Amy：Revenue share 至 2030 可預測性佳；RF share 取消有利毛利率。
Section 07

投資關鍵論點

多頭論點（Bulls）
  • Copilot 20M+ 席位加速，usage intensity = Outlook，TAM 從 seat → seat + consumption 擴大
  • Azure 39% CC 持續加速且 H2 CY26 有望再升
  • AI 業務 ARR $37B (+123%)，已證明 AI 可同時擴張營收與利潤
  • CapEx 低於預期反映效率提升，非需求疲弱；Cloud GM 66% beat
  • OpenAI IP 免版稅至 v32，結構性毛利率優勢
  • FY26 OI margin +1pp YoY，FY27 再承諾雙位數成長
空頭論點（Bears）
  • CY2026 CapEx ~$190B（含 $25B 漲價），FCF 壓力持續
  • Windows OEM Q4 預計 high teens 下降，PC 市場疲弱
  • Gaming −7%，結構性衰退風險
  • Commercial bookings -4%/-6% CC（含 OpenAI），合約週期不確定性
  • Cloud GM 趨勢 66%→64%（Q4），AI 投資長期壓力
  • OpenAI 不再獨佔 Azure，客戶分散風險浮現
關鍵觀察指標（Watch）
  • Q4 CapEx 是否能達 >$40B（供應鏈執行）
  • GitHub Copilot usage-based pricing 6/1 上線後的營收影響
  • Copilot 席位 QoQ 淨增是否再加速
  • Azure H2 CY26 加速幅度（FY27 指引具體化時點）
  • E7 定價 & 上市時間
  • VRP 後人員精簡幅度與效率成效
