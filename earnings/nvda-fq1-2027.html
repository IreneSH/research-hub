06 ｜ 中國出口管制風險
現狀：美國政府已批准 H200 出口至中國客戶的許可證，但 NVIDIA 尚未產生任何收入，亦不確定進口是否最終被允許。

Guidance 處理方式：與上季相同，Q2 Guidance ($91B) 明確不含任何中國 DC 計算收入。換言之，任何中國訂單均屬純粹上行選擇權 (upside optionality)，而非基準情境。

對推論的啟示：若中國最終獲准進口，潛在增量可觀；但若持續受阻，現有 Guidance 已完整反映此風險，不構成下修壓力。

08 ｜ Earnings Call 核心 Q&A 精華
  • 【新分部框架邏輯】（Morgan Stanley / Joseph Moore）
    Jensen 解釋分部重組反映 AI 應用的三層多樣性：①資料類型（語言、3D、蛋白質、物理）②應用場景（超大規模雲、AI 原生、企業/工業邊緣）③治理方式（公有雲 vs 主權 vs 機密運算）。新框架幫助市場更清楚理解各自的 go-to-market 邏輯
  • 【成長是否快於超大規模 CapEx？】（Melius / Ben Reitzes）
    Jensen 明確確認「理應如此」，因 ACIE 分部（AI 原生雲、企業、工業）的成長速度高於 Hyperscale，加上物理 AI 未來加入，均為獨立且額外的成長動能，非替代
  • 【推論市場份額】（Cantor Fitzgerald / CJ Muse）
    Jensen：「我們在推論的份額正在非常快速地增長。」關鍵因素：① 正式納入 Anthropic 作為策略夥伴，過去 coverage 幾乎為零；② Vera Rubin 將比 Blackwell 更成功，每一家前沿模型公司均計劃從一開始即採用；③ 非 Hyperscale 的第二類別 AI 資料中心，推論份額「幾乎 100%」來自 NVIDIA
  • 【LPX / CPX 定位】（UBS / Tim Arcuri）
    Jensen 重申 LPX（SRAM 架構）屬利基定位：低延遲高 token 速率，但吞吐量低、context 視窗小。現階段市場占比可能遠低於 20%；未來高端溢價 token 服務或達 20%。LPX 作為 Vera Rubin 平台的「加掛」選項，而非替代
  • 【CPU vs GPU 關係是否蠶食？】（BofA / Vivek Arya）
    Jensen 明確否認：Agent 架構中，「所有思考在 GPU，所有協調在 CPU」。Vera 定位為 Agentic CPU，類比每個 Agent 均需一台 PC；未來 agent 數量可能達數十億，形成全新 $200B TAM。$20B 為 standalone CPU 能見度，Vera Rubin 內含的 Vera 另計
  • 【$1兆之上的 Upside 來源排序】（Goldman Sachs / Jim Schneider）
    Jensen 排序：①前沿 AI 模型份額持續擴大（Anthropic 等新夥伴）②Vera 獨立 CPU（$1兆基準未含）③LPX。整合 Vera + Vera Rubin + LPX 可覆蓋完整 AI 全週期
  • 【Vera Rubin 爬坡速度】（TD Cowen / Joshua Buchalter）
    Colette：Q3 開始出貨（初始），Q4 加速，FY28 Q1 應相當大規模。客戶 PO 已到位，幾乎所有主要客戶準備就緒；系統極為複雜，速度取決於生產整合時程，非需求問題
  • 【ACIE vs Hyperscale 長期成長差異】（Bernstein / Stacy Rasgon）
    Jensen 確認 Neo Cloud（AI 原生雲）歸屬第二類別（ACIE）。預期 ACIE 長期成長仍快於 Hyperscale，原因是企業 / 工業市場代表 $50–80 兆全球經濟，AI 滲透率仍極低；物理 AI 的第三類別有望在 5 年內快速成長

10 ｜ 關鍵觀察與投資含義
  • 成長引擎從「Hyperscale 單引擎」轉向「雙引擎」：ACIE 本季 QoQ 成長 31% 遠快於 Hyperscale 的 12%，若持續，NVIDIA 的成長將不再完全依賴超大規模 CapEx 週期，結構性多元化程度提升
  • Vera CPU 是本季最大意外驚喜：$20B 年度 CPU 能見度為市場首次明確量化，開啟 NVIDIA 進軍 $200B CPU TAM 的新敘事，長期可挑戰 Intel / AMD 企業 CPU 市場
  • Anthropic 夥伴關係重要性被低估：Call 中多次強調過去對 Anthropic 的 coverage「幾乎為零」，現在正在快速補量，AWS / Azure / CoreWeave 等多管道佈局，代表短期推論需求的重要增量
  • Q2 Guidance 是近期最重要的催化劑：$91B 中值高於 consensus $87.3B 約 4%，且毛利率無下修，排除了近幾季最大隱憂；本次 guidance beat 幅度為近年最大
  • 股價不漲、甚至回跌的反直覺現象延續：NVIDIA 已連續數季「超預期後股價下跌」，反映買方 whisper number 慣性高於 consensus；然而 Q2 guidance 明顯超預期，此次後續走勢或有所不同，值得觀察
  • 毛利率短期天花板已相對清晰：Non-GAAP 75% 為管理層全年目標上限，Blackwell/Rubin 系統複雜度使進一步提升受限；供應鏈成本壓力（OpEx 全年成長 upper 40s）需持續監控
